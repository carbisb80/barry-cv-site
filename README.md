# Barry Carbis - Quality Engineering Leader CV Site

A Next.js website tailored to position Barry Carbis as a senior leader in software quality engineering.

## Stack

- Next.js 16.2.1
- React 19.2.1
- React DOM 19.2.1
- TypeScript 5.9.2

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Push this folder to a Git repository.
2. Import the repository into Vercel.
3. Vercel will detect Next.js automatically.
4. Use the default build command: `next build`
5. Use the default output setting provided by Vercel.

## Notes

- Uses the App Router.
- No environment variables are required.
- Content is written to present a leadership-focused quality engineering profile.
